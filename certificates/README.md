# Certificates Folder

Place your certificate files here.

## Instructions:
1. Save your certificates as PDF files
2. Name them as:
   - `certificate1.pdf` - Your first certificate
   - `certificate2.pdf` - Your second certificate
   - `certificate3.pdf` - Your third certificate

3. Update the titles and descriptions in `index.html`:
   - Find the Certificates section (around line 400)
   - Update the `<h3>` tags with your certificate names
   - Update the `<p>` tags with descriptions

4. The view links will work automatically!

**Note:** You can add more certificate cards by copying the existing card HTML structure.
